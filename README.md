# Data

The dataset used in this project is publicly available and should be downloaded before running the notebook.

## Download

Run the following command from the project root:

```bash
curl -o data/world_happiness_2005-2021.csv \
  https://raw.githubusercontent.com/csbfx/cs133/main/world_happiness_2005-2021.csv
```

Or manually download from:  
https://raw.githubusercontent.com/csbfx/cs133/main/world_happiness_2005-2021.csv

## About

- **Records:** 2,089 (1,910 after preprocessing)
- **Years covered:** 2005–2021
- **Source:** Gallup World Poll / World Happiness Report
